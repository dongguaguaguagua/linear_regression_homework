## 构建环境

本repo使用renv，故请先安装renv
```R
install.packages('renv')
```

cd进目录可自动激活renv环境，并且下载所需的包。

如果没有可手动安装:

```R
renv::install('languageserver','ggplot2','httpgd','jsonlite','rmarkdown','showtext', 'readxl')
```
